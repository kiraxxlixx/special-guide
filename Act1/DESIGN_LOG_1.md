# Bitácora de Diseño UX - Actividad 1

## 1. Prompt Utilizado
> "Crea una Landing Page HTML para una app de mapas llamada TijuArte Urbano. Debe tener un 'Hero' con una imagen de fondo de un mapa estilizado o topográfico, un título grande, y un botón CTA prominente que diga 'Explorar Mapa'. Usa Tailwind CSS. El diseño debe inspirar aventura/seguridad. "
## Cambios a resultados
>"Está muy bien, pero faltó que se utilice la imagen de fondo de un mapa estilizado o topográfico"
>"Pienso que sigue faltando el toque de mapa, algo geográfico que represente una app de mapas"

## 2. Decisiones de Diseño
* **Paleta de colores:** Se usó una paleta oscura (Slate-900) con acentos vibrantes (Fuchsia/Cyan) que recuerdan a los colores neón y al arte callejero de Tijuana.
* **Metáforas de Mapa y Exploración:** Para lograr que se sienta como una "App de Mapas" y no solo una galería de arte, se usaron estos elementos:
Líneas Topográficas (SVG Animado): Las curvas blancas gigantes que flotan en el fondo representan las isolíneas de los mapas geológicos (elevación del terreno). Inspiran aventura y terreno físico.

Grid Táctico (Cuadrícula): La cuadrícula fina (bg-grid-map) imita la interfaz de un GPS militar o técnico. Sugiere precisión y tecnología de localización.

Vista Satelital Oscura: El fondo no es negro sólido, sino una foto aérea de ciudad muy oscura. Esto simula el "Modo Noche" de los navegadores GPS modernos.

Coordenadas (HUD): El texto pequeño LAT: 32.5149° N imita un "Heads-Up Display" (pantalla de datos), anclando el diseño en la realidad geográfica de Tijuana.
